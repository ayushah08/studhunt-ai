AI-powered student roadmap & study assistant.

## 🔥 Problem
Students don’t know:
- what to study
- how to manage time
- how to revise effectively

## 💡 Solution
StudHunt AI helps students by:
- Generating personalized roadmaps
- Converting notes → MCQs / summaries
- Helping with study planning
- Predicting exam patterns (future scope)
